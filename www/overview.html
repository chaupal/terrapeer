Project Overview

Keywords

Distributed Virtual Environments (DVE), Peer-to-Peer (P2P), 3D User Interfaces, Avatars, Virtual Spaces and Zones, Distributed Networks, Online Games and Worlds, Cyberspace, Services, Web, User Rights, Trust and Access

Goals

The main goal is to create an application that supports a decentralized network, and virtualization in 3D space. The application should be able to connect to a peer-network, and to display a virtual world that users can move through, and meet other users.

Other possible requirements:

  • the ability to create virtual 'land' (zones), which would require standards for dynamic allocation of space (reservation system)
  • access control of virtual objects
  • to support the exchange of data between peers
  • pluggable interface for 3rd party services (web-browser, chat, file-sharing, forum, etc.)
  • enable the user to manage personal information
  • user-representation by avatars
  • group management
  • a peer trust system
  • the extension of the levels

Design

The application is split into several main layers that represent the group of objects which are responsible for the peer-network connectivity, the peer base, the virtual 3D world, respectively.

6. GUI Layer - the 2D user interface
5. Virtual User Interface (VUI) Layer - 3D display and control
4. Service Layer - sercices/plugins
3. Core Layer - logic
2. Repository - storage system
1. Peer Network Layer - JXTA dicovery, connectivity etc.

Layer 1 will use the JXTA protocols, which provides connection-, messaging-, group-, and security-services. The network layer will be abstract to be able to run any other service on level 3. It's main purpose will be to connect to other peers, exchange data, and manage groups and security.

Level 5 draws upon the information and mechanisms provided in layer 4, while it keeps track of 3D-world related data. This data includes information about the peer's own and other's virtual 'homes' and positions, and requires some storage system. The virtual world service level is responsible for the 3D user interface, a basic world, navigation support, as well as a standard set of rights & rules.
