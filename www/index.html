P2P Based Distributed Virtual Environment

Abstract

This project aims to build an architecture and implementation of a distributed virtual environment application, called TerraPeer. The conceptual idea is to design an interface to a multi-user virtual space, which runs on a server-independent network. TerraPeer is build with specific technological choices, including a user interface framework on top of the Java3D API, and the JXTA peer-to-peer platform. Examining existing systems and current research in relation to three-dimensional virtual reality and distributed networks, this project aims to assemble the various parts that are required to create such an interface to cyberspace.

Keywords
